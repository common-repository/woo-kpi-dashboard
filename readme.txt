Woo-KPI Dashboard is a Wordpress plugin to analyze your KPI and give you all the informations to help you maximize your income.

== Description ==

Woo-KPI Dashboard is a Wordpress plugin for Woocommerce users.
With its clear and ergonomic interface, Woo KPI Dashboard gives you access to all the informations needed to reach the objectives you have defined: number of unique visitors, conversion rate, average basket, and more. All the informations are also displayed via a chart allowing you to study your KPI's evolution throughout the year.
On top of that, Woo KPI Dashboard analyzes your results to give you various tips to increase you income and thus reach or even surpass your objectives


== Installation ==

1. Upload the plugin directory 'woo-kpi-dashboard' to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the Woo-KPI -> Analytics screen
4. Click "Click here to Authenticate"
5. Authorize the access to Analytics data by the plugin
6. Copy the Access Code it gives you and paste it in the "Your Access Code" field
7. Click "Save Changes"
8. Go to the "Profile" tab and select which website you want the analytics for.
9. Click "Save Changes"
10. Go to the Woo-KPI -> Objectives screen
11. Enter your objectives for each month
12. If you have bought the Pro Version, enter your license in the "License Key" field and click the "Activate License" button
13. Click "Save Changes"
14. You can now access your KPI on the  Woo-KPI -> Dashboard page


== Frequently Asked Questions ==

=Do I need to have Woocommerce installed on my website for this plugin to work?=

Yes, Woo-KPI Dashboard only works on websites that use Woocommerce.

=Does my website need to use Google Analytics for this plugin to work?=

Yes, Woo-KPI Dashboard only works on websites that use Google Analytics.
